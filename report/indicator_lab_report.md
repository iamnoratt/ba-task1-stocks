# Indicator Lab 报告

## 1. 任务目标

本次任务的目标是用白盒方式完成一次指标分析实验。所谓白盒，指的是不仅要展示最终图表，还要明确保留原始数据、公式定义、计算过程、参数设置以及可复现的代码结构。

## 2. 数据说明

- 标的 1：中国巨石（600176）
- 标的 2：中际旭创（300308）
- 数据来源：沿用 Task1 中获取的前复权日线数据
- 原始数据位置：`raw_data/`
- 时间范围：请根据实际读取结果补充

## 3. 指标设计

本次实验建议重点分析以下指标：

1. 移动平均线（MA5 / MA10 / MA20 / MA60）
2. MACD（12, 26, 9）
3. 布林带（20, 2）
4. 振幅
5. RSI（14，可选但推荐）

## 4. 白盒流程说明

### 4.1 原始数据读取

说明你如何读取原始 CSV，以及原始数据包含哪些字段。

### 4.2 数据清洗

说明你做了哪些基础处理，例如：

- 日期格式转换
- 数值列转换
- 排序检查
- 缺失值检查

### 4.3 指标计算

请分别说明：

- MA 是如何计算的
- MACD 是如何计算的
- 布林带是如何计算的
- RSI 是如何计算的
- 振幅是如何计算的

### 4.4 图表结果

建议逐图写观察：

- 价格 + MA + 布林带
- RSI
- MACD
- 振幅

注意：这里写“客观描述”，不要直接写买卖结论。

## 5. 主要观察

你可以从这些角度写：

- 当前价格位于均线上方还是下方
- 是否接近或突破布林带
- MACD 红绿柱有什么变化
- RSI 是否接近超买或超卖
- 最近波动是否明显放大

## 6. 为什么这是白盒分析

这一部分建议重点写：

- 我保留了原始数据
- 我明确了公式和参数
- 我把每一步计算拆开了
- 我可以修改参数并重新运行
- 我可以把这套流程复用到其他股票

## 7. HTML 工具设想

结合老师截图，这个工具希望实现：

- 左侧参数卡片
- 右侧主图和副图联动
- 股票切换
- 时间区间切换
- BOLL / RSI / MACD 参数可调整

## 8. 后续改进方向

后续可以继续增强：

- 比较不同参数结果
- 比较不同标的
- 增加回测模块
- 增加交易信号标记
